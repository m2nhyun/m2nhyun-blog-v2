# m2nhyun-blog-v2

김민현의 기술 블로그이자 포트폴리오 사이트입니다.

문제 정의, 선택한 접근, 결과를 짧고 분명하게 남기는 데 초점을 둡니다.

## Live

- Site: https://m2nhyun-blog-v2.vercel.app
- GitHub: https://github.com/m2nhyun/m2nhyun-blog-v2

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Supabase
- Vercel

## What It Shows

- 기술 블로그
- 프로젝트 정리
- 관리자 글 작성
- Supabase 기반 데이터 관리

## Local

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

## Verify

```bash
pnpm verify
```
